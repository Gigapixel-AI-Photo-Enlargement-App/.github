## 🚀 What It Is
Gigapixel AI for Windows is Topaz Labs' answer to the age-old problem of enlarging a photo without turning it into a soft, pixelated mess. Where ordinary resizing simply spreads existing pixels further apart, Topaz Gigapixel AI studies the image and reconstructs the detail that a bigger canvas needs. The result is a larger picture that still looks crisp — ideal when you need to crop tightly, print at scale, or breathe new life into older, lower-resolution shots.

## ✨ Key Features
- 📈 **Multi-factor enlargement** — Scale images up by 2x, 4x, 6x, or beyond while the upscaler keeps edges clean and defined.
- 🎨 **Realistic detail synthesis** — AI models add believable texture to skin, hair, foliage, and surfaces instead of just smoothing them.
- 🗂️ **Batch processing** — Point Topaz Gigapixel AI at a whole folder and upscale dozens of images in one unattended pass.
- 🔌 **Standalone or plugin** — Use it as a self-contained app or as an image upscaling plugin inside your regular editing pipeline.
- 💾 **Export flexibility** — Set precise output dimensions and save in the formats you need for web or print.

> With Gigapixel AI you can turn a modest source file into a large, sharp image — the Topaz Gigapixel AI upscaler focuses on recovering the detail an enlargement would otherwise lose.
